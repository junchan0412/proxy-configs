# Surge Policy Group Icons

透明小尺寸 Surge 策略组图标资产。每个图标提供 SVG 母版和 144 x 144 PNG，可直接作为 `icon-url` 使用，也可通过图标集 JSON 导入 Surge：

```text
https://fastly.jsdelivr.net/gh/qidewei2004/proxy-configs@main/icons/surge-policy-groups/policy-icons.json
```

设计参考：

- Qure mini：透明背景、单色粗线、小尺寸优先。
- Qure Light：透明 PNG 图标集结构。
- ColorfulStaticFlag：地区策略组使用圆角旗帜，而不是再包一层深色底板。
